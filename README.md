## Untitled Tab App

A web app created because my friend bet me $15 to make it and learn Supabase. He also wanted me to teach my friend how to deploy the website on Vercel.

Untitled Tab App is tool that allows you to keep track of your stingy friends who won't pay you back after you decided to be nice and pay for dinner.

## Features

- Create and remove different tab profiles.
- Create and remove friends.
- Mark people as paid.
- Calculate payments, tip, and totals.
